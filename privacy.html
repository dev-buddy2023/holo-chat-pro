最后更新：2026 年 6 月 24 日

1. Introduction

1. 引言

This Privacy Policy describes how Holo Chat Pro ("we", "our", or "the app") handles your information when you use our iOS application. Holo Chat Pro is designed with privacy as a core principle — all sensitive data stays on your device. We have no servers, no databases, no cloud infrastructure, and no access to your conversations.

本隐私政策说明了 Holo Chat Pro（"我们"或"本应用"）在使用 iOS 应用时如何处理您的信息。Holo Chat Pro 以隐私为核心设计原则——所有敏感数据均存储在您的设备上。我们没有服务器、数据库、云基础设施，也无法访问您的对话内容。

By using Holo Chat Pro, you agree to the practices described in this policy.

使用 Holo Chat Pro 即表示您同意本政策所述的做法。

2. Information We Collect and How We Use It

2. 我们收集的信息及使用方式

2.1 API Keys

2.1 API 密钥

  • What: Your API key(s) from OpenAI, OpenRouter, or any compatible provider, entered manually in Settings.
  • 内容：您在设置页面手动输入的 OpenAI、OpenRouter 或任何兼容提供商的 API 密钥。
  • Storage: Stored securely in the iOS Keychain (hardware-backed AES-256 encryption). It never leaves your device except when making authorized requests to your chosen API provider.
  • 存储方式：安全存储在 iOS Keychain 中（硬件级 AES-256 加密）。仅在向您选择的 API 提供商发起授权请求时才会离开���的设备。
  • Usage: Used exclusively to authenticate API calls to your provider for model access and conversation processing.
  • 用途：仅用于验证对 API 提供商的身份认证，以访问模型和处理对话。
  • Control: You can add, switch, or delete API keys at any time through the app settings. Keys support multiple profiles for different providers.
  • 控制权：您可以随时在应用设置中添加、切换或删除 API 密钥，支持多个提供商配置文件。

2.2 Conversations & Messages

2.2 对话与消息

  • What: Your chat messages, including text, images, voice memos, and video files.
  • 内容：您的聊天消息，包括文本、图片、语音备忘录和视频文件。
  • Storage: Each conversation is saved as an individual file on your device. You can browse, copy, back up, edit, or transfer them through the iOS Files app.
  • 存储方式：每条对话以独立文件保存在您的设备上。您可以通过 iOS 文件应用浏览、复制、备份、编辑或传输。
  • Processing: Messages are sent to your chosen API provider for AI response generation. We do not have access to this data.
  • 处理方式：消息发送给您选择的 API 提供商以生成 AI 回复。我们无法访问这些数据。
  • Recycle Bin: Deleted conversations are moved to the Recycle Bin and retained for 30 days before permanent deletion. You can recover or permanently delete them at any time.
  • 回收站：已删除的对话进入回收站，保留 30 天后永久删除。您可以随时恢复或永久删除。

2.3 Voice Input

2.3 语音输入

  • What: When you record a voice memo, your spoken words are captured as audio.
  • 内容：录制语音备忘录时，您的话语以音频形式被捕获。
  • Processing: Audio is transcribed to text entirely on your device using iOS Speech Recognition framework. The raw audio file is saved locally and only sent to your API provider if you choose to include it in a message.
  • 处理方式：音频完全在设备本地通过 iOS 语音识别框架转写。原始音频文件保存在本地，仅当您选择将其作为消息发送时才传给 API 提供商。
  • Retention: Voice memos you attach to messages are kept as part of the conversation file on your device until you delete them.
  • 保留期限：附加到消息中的语音备忘录作为对话文件的一部分保存在您的设备上，直到您删除。

2.4 Model Data (WebUI Browser)

2.4 模型数据（WebUI 浏览器）

  • What: Information about available LLM models (names, capabilities, pricing, context windows, tool support, etc.) from OpenRouter and other providers.
  • 内容：来自 OpenRouter 和其他提供商的可用 LLM 模型信息（名称、能力、价格、上下文窗口、工具支持等）。
  • Source: Loaded from the provider's API when you browse models. No personal information is included in these requests beyond your API key.
  • 来源：浏览模型时从提供商的 API 加载。这些请求中除 API 密钥外不包含任何个人信息。

2.5 Project Files

2.5 项目文件

  • What: When you link a conversation to a local project folder, the AI can read, write, and manage files in that folder.
  • 内容：当您将对话关联到本地项目文件夹时，AI 可以读取、写入和管理该文件夹中的文件。
  • Access: Only happens with your explicit permission on a per-conversation basis. Files are sent to your API provider only when the AI tool requires it.
  • 访问：仅在您逐条对话明确允许的情况下进行。文件仅在 AI 工具需要时才会发送给 API 提供商。

2.6 User Preferences

2.6 用户偏好

  • What: App theme (light/dark/system), accent color, language selection, app icon choice, color tags for conversations, and widget preferences.
  • 内容：应用主题（浅色/深色/系统）、主题色、语言选择、应用图标、对话彩色标签和小组件偏好。
  • Storage: Stored locally in UserDefaults on your device. Not synced to any cloud service.
  • 存储方式：本地存储在���备的 UserDefaults 中，不同步到任何云服务。

3. Third-Party Services

3. 第三方服务

Holo Chat Pro communicates with one or more external services that you explicitly configure:

Holo Chat Pro 与您明确配置的一个或多个外部服务通信：

  • Your chosen API provider (e.g., OpenAI, OpenRouter) — receives your messages, API key, and any attached files for AI response generation.
  • 您选择的 API 提供商（如 OpenAI、OpenRouter）——接收您的消息、API 密钥和附加文件以生成 AI 回复。

Each provider's own privacy policy applies to data processed on their servers. We encourage you to review them:

各提供商自身的隐私政策适用于其在服务器上处理的数据。建议您查看：

  • OpenRouter Privacy Policy
  • OpenAI Privacy Policy

4. No Tracking, No Analytics, No Ads

4. 无追踪、无分析、无广告

  • No analytics SDKs — we do not use Google Analytics, Firebase, Amplitude, or any similar service.
  • 无分析 SDK——我们不使用 Google Analytics、Firebase、Amplitude 或任何类似服务。
  • No crash reporting — we do not use Crashlytics, Sentry, or any crash reporting tool.
  • 无崩溃报告——我们不使用 Crashlytics、Sentry 或任何崩溃报告工具。
  • No advertising — there are no ads in the app and no ad-related SDKs.
  • 无广告——应用内无广告，也不含广告相关 SDK。
  • No third-party trackers — we do not embed any tracking pixels, fingerprinting scripts, or behavioral profiling tools.
  • 无第三方追踪器——我们不嵌入任何追踪像素、指纹脚本或行为分析工具。

5. Data Security

5. 数据安全

  • API Keys: Stored in iOS Keychain with hardware-backed AES-256 encryption.
  • API 密钥：存储在 iOS Keychain，硬件级 AES-256 加密。
  • Local files: Conversation files are stored in the app's sandboxed Documents directory. No other app can access them without your permission.
  • 本地文件：对话文件存储在应用沙盒的 Documents 目录中，其他应用未经许可无法访问。
  • Network: All communication with API providers is over HTTPS/TLS 1.3.
  • 网络：与 API 提供商的所有通信均通过 HTTPS/TLS 1.3。
  • No backend: Holo Chat Pro has no server infrastructure — there is no database, no cloud storage, and no backend service operated by us.
  • 无后端：Holo Chat Pro 没有服务器基础设施——我们运营的没有数据库、没有云存储、没有后端服务。

6. Children's Privacy

6. 儿童隐私

Holo Chat Pro is not directed at children under the age of 13 (or the applicable age of consent in your jurisdiction). We do not knowingly collect personal information from children. If you believe a child has provided us with personal data, contact us immediately and we will take steps to delete such information.

Holo Chat Pro 不面向 13 岁以下儿童（或您所在司法管辖区的适用同意年龄）。我们不会故意收集儿童的个人信息。如果您认为有儿童向我们提供了个人数据，请立即联系我们，我们将采取措施删除此类信息。

7. 隐私政策的变更

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated "Last updated" date. Material changes will also be communicated through the app.

我们可能会不时更新本隐私政策。变更将通过此页面发布，并更新"最后更新"日期。重大变更也会通过应用内通知告知。

8. 您的权利（GDPR / CCPA / 其他司法管辖区）

Since Holo Chat Pro does not collect or store personal data on any server:

由于 Holo Chat Pro 不在任何服务器上收集或存储个人数据：

  • Right to Access / Deletion: There is no server-side data to access or delete. You can manage all local data (API Keys, conversations, preferences) directly within the app or through the iOS Files app.
  • 访问/删除权：没有服务器端数据需要访问或删除。您可以在应用内或通过 iOS 文件应用直接管理所有本地数据（API 密钥、对话、偏好设置）。
  • Right to Object: You can stop using the app at any time. No ongoing data processing occurs after you close the app.
  • 反对权：您可以随时停止使用本应用。关闭应用后不会继续处理任何数据。
  • Data Portability: All your data (conversations, settings) is already under your control on your device. Conversations are individual files you can copy freely.
  • 数据可移植性：您的所有数据（对话、设置）已在您的��备上由您自行控制。对话是您可以自由复制的独立文件。
